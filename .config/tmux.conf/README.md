# Tmux
## Plugin manager:
### Tmux and Tpm, with spotify setup
```
sudo apt install tmux
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
cd ~
git clone https://github.com/polthelost/tmux-spotify
```
